# Placeholder Images

This directory contains placeholder images for the ASL Jeux Écossais website.

## Structure

```
public/
├── images/
│   ├── home/          # Homepage images
│   ├── events/        # Event photos
│   ├── prestations/   # Service demonstration photos
│   ├── sponsors/      # Sponsor logos and partnership photos
│   └── placeholder.jpg
└── videos/
    └── hero.mp4       # Hero section video
```

## Image Guidelines

### Homepage
- **Hero video**: 1920x1080px, MP4 format
- **Gallery photos**: 1200x900px (4:3 ratio), showcasing Highland Games demonstrations

### Prestations
- **Service photos**: 1200x900px, showing:
  - Festival demonstrations
  - Municipal events
  - Team building activities
  - Corporate events

### Sponsoring
- **Partner logos**: 400x200px, PNG with transparent background
- **Event photos**: 1200x900px, showing sponsor visibility at events

## To Add Your Images

1. Place your images in the appropriate subdirectory
2. Use descriptive filenames (e.g., `festival-luzarches-2024.jpg`)
3. Optimize images before upload (compress, resize appropriately)
4. Update the corresponding data files to reference your images

## Placeholder

Until you add your own images, the gallery will use a placeholder pattern.
